About
DJ Booth Lite with optional DJ Booth Pro integration. Main screen remains performance-only; setup lives behind this cog.
How This Works
Built for TommyWho Records · Lite playback uses clean Web Audio raw playback on hosted MP3 files. BPM & Key pulled from the Serato-analyzed catalog. Real 3-band EQ, real waveform (drawn from the decoded audio), and real channel fader volume control. BPM Sync changes raw playback speed, so pitch moves naturally with tempo. Key Lock and Key Sync are reserved for DJ Booth Pro. Load tracks by pasting a direct MP3 URL, or via the browser once mp3Url is filled in per track. Auto Cue detects the first beat on load and skips leading silence — hit SET CUE any time to override it manually. USB MIDI controllers work via the MIDI tab above — connect, click LEARN next to a control, then press/move the matching button or fader on your hardware.
